Set Up in 15 Minutes

Do this ONCE. Then every day: start VM → learn → stop VM → go to sleep.

☁️ Create GCP Account
1
Go to cloud.google.com/free → click Get started for free
2
Sign in with Google → Country: India → Add card (verification only — not charged)
3
You get $300 free credit valid 90 days — enough for this entire journey
4
Go to console.cloud.google.com → you're in!
🖥️ Create Forge VM
# Run this in GCP Cloud Shell
$ gcloud compute instances create forge-playground-vm \
  --machine-type=e2-standard-2 \
  --image-family=ubuntu-2204-lts \
  --image-project=ubuntu-os-cloud \
  --boot-disk-size=50GB \
  --zone=us-central1-a \
  --tags=http-server,https-server

# Takes 60 seconds. VM is ready!
💻 Connect VS Code
1
Install Remote-SSH extension in VS Code on your Mac
2
Press Cmd+Shift+P → Remote-SSH: Connect to Host → add new host
3
Type: ssh -i ~/.ssh/forge-key YOUR-VM-IP
4
New VS Code window opens — you're coding on the cloud VM!
⚙️ Install All Tools
# Run ONCE on your VM terminal
$ curl -fsSL https://get.docker.com | sudo sh
$ sudo snap install terraform --classic
$ sudo apt install ansible git -y
$ curl https://raw.githubusercontent.com/helm/helm/main/scripts/get-helm-3 | bash

✅ All tools ready. Start Mission 01!
